## Installation

```
//Install Node If you don't have on your machine.

//Install yarn
npm install -g yarn

//Install dependencies
cd [download path]/frontend-main
yarn
```

## Usage

Start server to check build files.

```
cd [download path]/frontend-main
yarn start-dist
```
